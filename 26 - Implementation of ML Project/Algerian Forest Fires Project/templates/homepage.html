Algerian Forest Fire Predictor

Machine Learning Based Fire Risk Forecasting

About the Project

This project predicts the likelihood of a forest fire in the Algerian region using meteorological and environmental data such as temperature, humidity, wind, and the Fire Weather Index (FWI). The model is trained on real Algerian forest fire datasets and provides insights to help authorities take preventive actions.

How It Works

  • Upload or input environmental data.
  • Model analyzes features using trained regression algorithms.
  • Predicts fire risk and provides confidence scores.
